## Tech Stack

- **Framework:** Next.js 15 + TypeScript  
- **Styling:** Tailwind CSS v4  
- **Deployment:** Vercel  
- **Icons:** Lucide  
- **Animations:** Framer Motion  

---

## Features

- **Clean & Modern Design** 
- **SEO Optimized** 
- **PWA Ready** 
- **Dynamic OG Images**: For rich social previews  
- **Component Registry**: Easily showcase and document reusable components
